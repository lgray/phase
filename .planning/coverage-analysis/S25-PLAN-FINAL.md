### Identity / Provenance Contract (mandatory per-card for provenance cards)
Each below pins: source phrase ‚Üí authority type & id/value ‚Üí binding time ‚Üí live-vs-latched ‚Üí storage ‚Üí consuming function ‚Üí invalidation ‚Üí multi-authority hostile fixture.

- **Rhino's Rampage ‚Äî "excess damage‚Ä¶ *this way*"**: authority = the specific damage event this spell dealt; bind at the spell's damage resolution; latched to that event id; consumed by the `DealtExcessDamage` reflexive trigger; invalidated after the trigger resolves. Hostile: a *second* unrelated excess-damage event same turn must NOT re-fire this trigger.
- **Prishe's Wanderings ‚Äî "search *this way*"**: authority = the search event this effect performed; latched at resolution; consumed by the reflexive put-counter; hostile: an unrelated library search does not add the counter.
- **Deadly Cover-Up / The End ‚Äî "cards with *that name* / the same name"**: authority = the name of the exiled/targeted card, **latched at resolution** (CR 201.3a); stored as the matched name; consumed by the multi-zone search filter; invalidated after the effect. Hostile: a same-name card entering a searched zone *after* latch is still matched by the latched name (name is a value, not a live pointer); a differently-named card is NOT exiled.
- **Foraging Wickermaw ‚Äî "become *that color*"**: authority = the mana color chosen when the ability's mana was added; bind at mana-add; stored on the `SetColor` mod's `colors`; live for the "until end of turn" duration. Hostile: adding a *different* color next activation sets a different color; SetColor from a prior activation is a separate timestamped effect.
- **Niko ‚Äî "copies of *it* until the next end step" (C3)**: authority = Niko's characteristics at effect start (CR 611.2c set fixed); duration-bound to next end step; consuming = `BecomeCopy` mass application. Hostile: Niko changing after the effect begins does not change the already-fixed copy set.
